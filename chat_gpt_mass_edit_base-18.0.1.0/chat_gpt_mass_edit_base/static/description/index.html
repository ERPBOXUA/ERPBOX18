# Configuration

How to configure

Step 1 Download the module and place the module in apps folder.

Step 2 Install this module in your Odoo instance.

Step 3 Open Chat GPT. If you want to use the ChatGPT API, you need to create an account on ChatGPT and get the API key.

Step 4 Register at https://platform.openai.com/

Step 5 Search your default project number

Step 5 Search your organization number

Step 5 Register a new API key

Go to Variants or Product Template menu. In the Actions you can choose "Chat GPT Mass Edit"

Go to In the wizard you can reference all fields from the model to get the query working for all selected entries. Even related Fields get resolved. You can then select to which field the result should be written to. All writeable fields are available.

Image Analysis: You can select any image in the model. Depending on the image images with lower resolution result in better results and use less tokens. The image and the query get evaluated from ChatGPT. You can ask GPT any question about the image or let GPT create a description of the product. Under "Field to map answer to" you can select any field to write the result. The default is the description field for ecommerce.

Image Creation: With the dynamic promt you can ask GPT to create an image for you. There is a special syntax to include the attributes {block_attributes}. Other fields can be included with {field_name}. The images can be stored in any field or in the extra images available in product.product or product.template.
